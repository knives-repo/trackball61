This series layout references: [keyball](https://github.com/Yowkees/keyball)<br/>
The firmware code is a secondary development based on the source code of [Charybdis](https://bastardkb.com), adding some practical functions.<br/>
<br/>
Versions with the suffix ".ctc" are dual Type-C connection versions, while those without ".ctc" are TRS connection versions.<br/>
<br/>
My keyboard suffix is "V3" and it uses the PMW3389 sensor<br/>
<br/>
To compile your own firmware, simply determine your sensor based on the keyboard suffix and then adjust the "POINTING_DEVICE_DRIVER" in v2/splinky/rules.mk for each layout.<br/>
